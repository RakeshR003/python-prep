# 📚 Python Lists

## 📖 Overview

A **List** is one of the most commonly used data structures in Python. It allows us to store multiple values inside a single variable. Lists are ordered, mutable (modifiable), and can contain duplicate values as well as different data types.

```python
students = ["Rakesh", "Rahul", "John"]
```

# 🛠️ List Methods Covered

| Method      | Purpose                                    |
| ----------- | ------------------------------------------ |
| `append()`  | Add an element to the end of the list.     |
| `insert()`  | Insert an element at a specific position.  |
| `remove()`  | Remove an element by its value.            |
| `pop()`     | Remove an element using its index.         |
| `len()`     | Return the number of elements in the list. |
| `count()`   | Count how many times a value appears.      |
| `index()`   | Return the first index of a value.         |
| `sort()`    | Sort the list in ascending order.          |
| `reverse()` | Reverse the order of the list.             |

# 💡 Key Points to Remember

* List indexing starts from **0**.
* Lists are **mutable**, so elements can be added, removed, or updated.
* `append()` always adds to the end.
* `insert()` places an element at a specified position.
* `remove()` removes by **value**.
* `pop()` removes by **index**.
* `len(list)` returns the number of elements.
* `len(string)` returns the number of characters.
* Use `sort()` only after collecting all data.
* Use `reverse()` only after collecting all data.
* Before using `index()` or `remove()`, check whether the value exists using `in`.

---

# ⚠️ Common Beginner Mistakes

* Forgetting that indexing starts from **0**.
* Confusing `remove()` with `pop()`.
* Calling `sort()` or `reverse()` inside the input loop.
* Using `index()` without checking whether the element exists.
* Forgetting to store user input before processing it.
* Confusing `len(name)` (letters) with `len(names)` (number of elements).

---

# 🧠 Memory Tricks

* **append()** → Add at the **end**.
* **insert()** → Insert at a chosen position.
* **remove()** → Remove by **value**.
* **pop()** → Remove by **position (index)**.
* **len()** → Count elements or characters.
* **sort()** → Arrange in ascending order.
* **reverse()** → Flip the order of the list.
